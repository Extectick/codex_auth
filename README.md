# ChatGPT Session Manager

Windows desktop-приложение на pywebview для хранения сессионных профилей ChatGPT и экспорта активного профиля в `auth.json`.

## Важно о безопасности

Профили сохраняются в открытом виде в `%APPDATA%\ChatGPTSessionManager\profiles\`. В MVP шифрование не используется. Не передавайте эти файлы третьим лицам и защищайте учетную запись Windows.

## Запуск из исходников

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m app.main
```

## Сборка Windows exe

```powershell
.\build.ps1
```

Готовый файл будет создан в `dist\ChatGPTSessionManager.exe`. Его можно переносить на другой компьютер одним файлом.

## Ручной импорт JSON

JSON должен содержать обязательные поля:

- `accessToken`
- `sessionToken`
- `account.id`
- `expires`

Имя профиля берется из `user.email`, затем `user.name`; если их нет, используется `Профиль N`. Поле `refresh_token` опционально.

## Экспорт

По умолчанию активный профиль записывается в `%USERPROFILE%\.codex\auth.json`. При выборе активного профиля файл сразу перезаписывается. Если активный профиль успешно обновлен, этот файл также перезаписывается автоматически.

В настройках экспорта можно выбрать другой путь и имя файла. Кнопка «По умолчанию» возвращает путь `%USERPROFILE%\.codex\auth.json`.

Истекший профиль нельзя сделать активным. Если ранее активный профиль истек, приложение сбрасывает выбор активного профиля и не изменяет существующий файл.
