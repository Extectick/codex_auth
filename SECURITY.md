# Security Policy

ChatGPT Session Manager stores profiles, tokens, exported session data, settings,
and logs locally in plain text. Treat `auth.json`, profile files, logs, and any
token-like values as secrets.

Do not attach `auth.json`, profile directories, screenshots with tokens, or logs
containing tokens to public issues or pull requests.

To report a security issue, open a private security advisory on GitHub if the
repository has advisories enabled. If advisories are not available, contact the
maintainer privately before publishing details.
